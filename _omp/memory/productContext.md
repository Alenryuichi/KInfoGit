# Product Context

## Why This Project Exists
<!-- 项目存在的原因，解决什么问题 -->

为 Kylin（苗静思）提供一个专业的在线简历和作品集平台，展示其在反欺诈技术、全栈开发和 UI 设计方面的专业能力和项目经验。

## Problems Solved
<!-- 用户面临的具体问题 -->

1. **专业形象展示**: 提供专业的在线简历替代传统 PDF
2. **项目成果可视化**: 清晰展示技术项目和量化成果
3. **技术博客平台**: 分享技术文章，建立技术影响力
4. **职业发展记录**: 集中管理职业规划和面试准备材料
5. **多语言支持**: 满足中英文展示需求

## User Experience Goals
<!-- 期望的用户体验 -->

- 🎯 快速加载，秒开体验
- 📱 响应式设计，任意设备完美展示
- 🌓 明暗主题切换，适应不同阅读环境
- 🔍 清晰的导航和信息架构
- ✨ 现代化 UI 设计，专业感强

## Target Users
<!-- 目标用户群体 -->

- **招聘方**: HR 和技术面试官
- **同行**: 技术社区成员
- **潜在合作者**: 寻找技术合作的人员
- **自己**: 职业发展记录和复盘

## How It Should Work
<!-- 产品应该如何运作 -->

1. **首页 (Hero)**: 第一印象，展示核心价值主张
2. **关于页面**: 技能矩阵和职业经历
3. **项目页面**: 详细项目案例和技术栈
4. **博客页面**: 技术文章列表和详情
5. **联系方式**: 便捷的联系渠道

## 内容管理

- 简历数据: `/profile-data/resume/`
- 项目数据: `/profile-data/projects/`
- 技能数据: `/profile-data/skills/`
- 博客文章: `/profile-data/blog/`
- 职业规划: `/profile-data/career/`
