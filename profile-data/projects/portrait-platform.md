## 项目概述

画像中台系统是腾讯反欺诈团队的核心基础设施，从 0 到 1 构建了支持 10 亿级用户规模的数据分析平台。

## 技术挑战

1. **海量数据处理**: 日均处理数据量超过 TB 级别
2. **实时性要求**: 标签计算需要准实时更新
3. **查询性能**: 复杂分析查询需要秒级响应

## 架构设计

### 数据采集层
- 支持多种数据源接入（Kafka、MySQL binlog、API）
- 统一的数据清洗和标准化流程

### 计算引擎层
- 基于 Spark 的批处理计算
- 基于 Flink 的实时流处理
- 自研的标签计算框架

### 存储层
- ClickHouse 作为分析型数据库
- Redis 作为热点数据缓存
- MySQL 存储元数据和配置

### 服务层
- RESTful API 对外提供服务
- 支持 200+ 标签维度的灵活查询
- 可视化分析界面

## 主要成果

- 支撑 200+ 标签维度的用户分析
- 查询响应时间 P99 < 3s
- 日活跃用户超过 500 人
- 被多个业务线复用，成为团队标准基础设施
